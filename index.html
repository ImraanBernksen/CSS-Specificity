A Example of CSS Specificity

CSS SPECIFICITY

What is Specificity?

If there are two or more CSS rules that point to the same element, the selector with the highest specificity value will "win", and its style declaration will be applied to that HTML element.Think of specificity as a score/rank that determines which style declaration is ultimately applied to an element.

Specificity Hierarchy:

Every CSS selector has its place in the specificity hierarchy.

There are four categories which define the specificity level of a selector:

  1. Inline styles (h1 style"color;pink;")
  2. IDs #( )
  3. Classes, pseudo-classes, attribute selectors .( )
  4. Elements and pseudo-elements (h1,:Before)

Inline style gets a specificity value of 1000, and is always given the highest priority. Internal styles are defined within the (style) element, inside the (head) section of an HTML page.

Equal specificity: the latest rule wins - If the same rule is written twice into the external style sheet, then the latest rule wins:

ID selectors have a higher specificity than attribute selectors

Contextual selectors are more specific than a single element selector - The embedded style sheet is closer to the element to be styled

A class selector beats any number of element selectors - a class selector such as .intro beats h1, p, div, etc
